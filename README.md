A modern, full-stack web application designed to bridge the gap between current students and university alumni. This platform facilitates seamless networking, allowing students to browse alumni profiles and request 1-on-1 mentorship sessions for career guidance.

##  Key Features

* **Role-Based Authentication:** Secure Login and Registration for both 'Student' and 'Alumni' roles using JWT (JSON Web Tokens).
* **Alumni Directory:** A dynamic, searchable database of alumni profiles showcasing their expertise and professional details.
* **Mentorship Request System:** Students can send personalized mentorship requests directly to alumni.
* **Personalized Dashboards:**
* **Students:** Track the status of sent requests (Pending, Accepted, Rejected) and access meeting links.
* **Alumni:** Manage incoming requests, accept/decline mentees, and provide session feedback.
* **Modern UI/UX:** Responsive, glassmorphism-inspired design with animated mesh gradient backgrounds.
